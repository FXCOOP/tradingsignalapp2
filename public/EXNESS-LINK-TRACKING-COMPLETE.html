🛠️ How to Use (Super Easy!)

1Import the Component

At the top of any file where you have Exness links:

import { ExnessLink } from '@/components/ExnessLink'
2Replace Regular Links
❌ OLD (Not tracked):
<a href="https://one.exnessonelink.com/a/c_8f0nxidtbt">
  Open Exness Account
</a>
✅ NEW (Auto-tracked):
<ExnessLink
  href="https://one.exnessonelink.com/a/c_8f0nxidtbt"
  source="broker_prompt"
>
  Open Exness Account
</ExnessLink>

📊 Attribution Reporting

After implementing, you'll be able to see:

In Supabase:
-- See which sources generate most clicks
SELECT
  click_source,
  COUNT(*) as total_clicks,
  COUNT(DISTINCT user_email) as unique_users
FROM exness_clicks
WHERE clicked_at >= NOW() - INTERVAL '7 days'
GROUP BY click_source
ORDER BY total_clicks DESC;

-- Results:
-- broker_prompt    | 145 clicks | 89 users
-- banner_signals   | 78 clicks  | 56 users
-- 30min_popup      | 34 clicks  | 34 users
-- footer_link      | 12 clicks  | 11 users
