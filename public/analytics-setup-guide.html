Configure Environment Variables

Add the credentials to your Render deployment settings.

Environment Variables to Add:

GA_SERVICE_ACCOUNT_EMAIL
Find "client_email" in JSON file
GA_PRIVATE_KEY
Find "private_key" in JSON file (entire key with -----BEGIN PRIVATE KEY-----)

Troubleshooting

‚ùå "Google Analytics Not Configured"

  • Check that all 3 environment variables are set in Render
  • Verify there are no extra spaces in the variable names
  • Make sure Render finished deploying after adding variables

‚ùå "Error: 403 Forbidden"

  • Service account doesn't have access to Analytics property
  • Go back to Step 5 and grant access
  • Make sure you added the correct service account email

‚ùå "Invalid property ID"

  • Check GA_PROPERTY_ID is the numeric Property ID, not Measurement ID
  • Don't include "G-" prefix (that's for Measurement ID)
  • Verify the Property ID in Google Analytics Admin > Property Settings

‚ùå "Error parsing credentials"

  • The private key likely has formatting issues
  • Make sure to keep the \n characters in the private key
  • Copy the entire private_key value from JSON including quotes
