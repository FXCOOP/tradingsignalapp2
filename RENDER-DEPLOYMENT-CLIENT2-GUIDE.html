    üîçTroubleshooting

    Build Failed

    Common Solutions:
    • Check build logs for specific error
    • Ensure package.json has correct scripts
    • Verify Node version compatibility
    • Try: Clear build cache & deploy

    Service Won't Start

    Check:
    • All required environment variables are set
    • Port is set to 10000 or using process.env.PORT
    • Start command is correct: npm start
    • Check runtime logs for errors

    Database Connection Issues

    Verify:
    • Supabase project is active
    • SUPABASE_URL and keys are correct
    • Database tables are created (run migrations)
    • Check Supabase dashboard for connection logs

    Custom Domain Not Working

    Steps:
    • Verify DNS records are correct (use dnschecker.org)
    • Wait up to 48 hours for DNS propagation
    • Check domain status in Render dashboard
    • Ensure SSL certificate is provisioned (automatic)
