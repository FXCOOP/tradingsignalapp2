Step 1: Update Main Page (page.tsx)

A. Pass Language to MultiPopupSystem

Find this line in your src/app/page.tsx:

<MultiPopupSystem onOpenBrokerAccount={handleOpenBrokerAccount} />

Change it to:

<MultiPopupSystem onOpenBrokerAccount={handleOpenBrokerAccount} language={language as 'en' | 'ar'} />

B. Add RTL Support to Body

Find your main <div> wrapper (around line 14,000+) and add dir attribute:

<div style={{...}} dir={language === 'ar' ? 'rtl' : 'ltr'} > {/* Your existing content */} </div>

C. Import Translation Hook

At the top of your src/app/page.tsx, add this import:

import { useTranslation } from '@/lib/translations'

Then inside your component function (after line 49 where language is defined), add:

const { t, isRTL } = useTranslation(language as 'en' | 'ar')
